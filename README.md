## Setup

### Install Dotnet Core
    - https://learn.microsoft.com/en-us/dotnet/core/install/macos

### Install Angular
    - https://angular.io/cli

## Clone Project

`git clone https://github.com/chelseapattee/Panda.Meow-temp.git`
    
## Run Server
    cd Panda.Meow.ai/
    cd ProjectServer/ 
    dotnet run

## Run Angular
    cd Panda.Meow.ai/
    cd PortfolioApp/ 
    npm install
    ng serve


## View App
     
http://localhost:4200/
